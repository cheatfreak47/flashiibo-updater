# Flashiibo Pro Updater Client
A hastily made electron update tool for [Flashiibo Pro](https://flashiibo.com/). Made with love for those who refuse to use anything other than FireFox.

## Installation
1. Download `flashiibo-pro-updater-vXXX-setup.exe` from Releases
   *(Portable version also available)*
2. Run installer (or run the portable EXE)
3. If installed, you'll have it in your Start Menu!
4. Follow the on screen instructions for updating your Flashiibo Pro device.

## Features
- Web Bluetooth support for Flashiibo hardware
- Automatic Flashiibo Pro device selection

## Build Instructions
1. Install [Node.js](https://nodejs.org/en/download)
2. Clone this repo: `git clone https://github.com/cheatfreak47/flashiibo-updater.git`
3. Open cmd in the repo location and install dependencies: `npm install`.
4. Open cmd in the repo location and build: `npm run build`.
